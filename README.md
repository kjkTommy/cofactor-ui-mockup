# КоФактор — UI mockup

Статический UI mockup под React + Next.js, собранный по мотивам присланного PDF-журнала.

## Что внутри

- `app/` — страницы App Router
- `components/` — общие блоки
- `public/assets/` — изображения, извлеченные из PDF и рендеров полос

## Страницы

- `/` — главная
- `/article` — статья
- `/category` — рубрика
- `/archive` — архив/содержание
- `/about` — о журнале
- `/contacts` — контакты

## Запуск

```bash
npm install
npm run dev
```

## Принципы макета

- максимально близкий стиль к печатному выпуску;
- мягкая editorial-композиция вместо типового блогового UI;
- адаптив: desktop + tablet + mobile;
- без логики и данных, только внешний вид.
